## Common Development Tasks

### Running the Application

- `yarn dev` - Run application

### Code Quality

- `yarn check-lint` - Check lint issues
- `yarn check-format` - Check code formatting
- `yarn check-types` - Check TypeScript types
- `yarn lint:fix` - Fix auto-fixable issues
- `yarn format` - Format code

After finishing with all edits, code must be ready for commit by checking: types, format and lint.

### Vite Testing

- `yarn test` - Run all tests
